About Visulization

The project follows the interactive slide show narrative visualization structure. The Home page represents the base of the visualization structure. The user is being exposed to the author driven content and will be led through three different scenes/slides with giving user ability to move backwards and forward as well as jumping directly to specfic slide by clicking on top navigation bar.

First slide is introductory slide and prepare user for visulization. Second slide gives overview of Breweries across united states. Third slide consider state population and shows number of breweries by 1 milion population by state to give user idea about where local brew and craft beers are more popular. Forth slide give different perspective to user by charting top 10 cities by total of number of breweries present in particular city.

Various slides use Axis name and tooltip for annotation and give user additional details.

The name of the target slide is passed as an input paramter to the button when user click on navgiation bar items.

The mouse click on navigation bar acts as a trigger event for navigation to different slides. Mouse scroll event take user to next or previous slide. The mouse hover (mouseover) event is a trigger for the tooltip in the bar chart.

The visulization is built using Vega-lite, D3 and bootstramp library. Dataset is obtained from www.kaggle.com and provided by Datafiniti.

    • Four different slides are used to give reader various perspective. Scenes are primilarily Author driven over mouse scroll event by user. However, Reader can jump to random scenes if he/she prefers.

    • Tooltip, change of colors and Axes names etc. are used as annotations for different scenes.

    Bar chart uses mouse position as parameter to mousehover event to display tooltip and chnage the other parameter color value.

    • The mouse click on navigation bar acts as a trigger event for navigation to different slides. Mouse scroll event takes user to next or previous slide. The mouse hover event is a trigger for the tooltip in all charts.
